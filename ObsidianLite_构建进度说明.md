# Obsidian Lite 构建进度说明

## 当前阶段

- 当前状态：原型可演示（Prototype）
- 目标状态：MVP 可持续使用（本地数据不丢失、关键交互闭环）

## 已实现的核心功能

### 1. 笔记库（Note Library）
- 笔记卡片列表展示
- 选中笔记后进入该笔记内容阅览
- 新建笔记入口（通过按钮进入独立创建页）

### 2. 笔记阅览与编辑流程
- 默认先阅览 Markdown 内容
- 用户主动点击“编辑笔记”后进入编辑态
- 可在编辑态完成标题、正文、标签修改
- 编辑完成后可返回阅览态

### 3. Markdown 轻量渲染（Preview Lite）
- 已支持：
  - 标题（#、##、###）
  - 列表（-）
  - 引用（>）
  - 行内加粗（**text**）
  - 行内代码（`code`）
  - 双链文本（[[link]])

### 14. Markdown 扩展
- 代码块（```）
- 任务列表（- [ ] / - [x]）
- 表格（| ... |）
- 图片（![](...)）

### 4. 标签与筛选
- 标签展示
- 标签筛选
- 对当前笔记添加新标签
- 分类展示与分类筛选

### 5. 全文搜索（基础）
- 支持按关键词检索：
  - 标题
  - 正文
  - 标签
  - 分类

### 13. 搜索增强
- 命中高亮（标题 / 分类 / 摘要 / 标签命中态）
- 相关性排序（标题 > 分类 > 标签 > 正文）
- 模糊匹配（连续命中、去空格匹配、子序列匹配）

### 11. 分类体系（Category）
- 支持笔记分类字段（创建 / 编辑 / 展示）
- 支持分类过滤与分类检索

### 12. 笔记管理闭环
- 删除笔记（详情页一键删除）
- 笔记置顶（固定在列表前排）
- 多维排序（按更新 / 标题 / 分类 / 最近访问）
- 最近访问快速入口（历史访问笔记快速跳转）

### 6. 双向链接相关能力（Lite）
- 正文中自动提取 [[...]] 链接
- 生成轻量图谱边关系
- 反向链接（Backlinks）面板展示

### 7. 图谱交互视图（Graph Interactive）
- 节点展示（笔记标题）
- 连线关系文本展示
- 双向关系标注
- 图谱画布支持拖拽浏览（横向/纵向）
- 支持图谱缩放（Zoom In/Out）
- 支持节点点击聚焦（邻居视图）与一键取消聚焦

### 8. UI 与布局
- 中文界面
- 暖纸张 + 卡片视觉风格
- 沉浸式阅读与编辑状态无缝切换
- 创建页覆盖层流程（点击后出现）

### 9. 本地持久化存储
- 基于 AppStorage 与 PersistentStorage 的本地核心数据持久化机制
- 实现了应用加载时读取以及修改后的实时自动保存

### 10. 双链无缝跳转与游历
- 点击 [[...]] 链接直接切换为对应目标笔记的内容闭环
- 支持识别不存在链接，并在点击时自动无缝创建关联空笔记
- 解决了 ArkUI 状态刷新、组件复用与列表渲染的多个底层兼容性与排障体验问题
